1. Go to [API Keys page](https://console.anthropic.com/settings/keys) on Anthropic.
2. Create a new key.
3. Paste the key into the `API Key` field in PreCallAI.
4. Write any screen name to be displayed in PreCallAI.
5. Click `Save`.
6. Start using Anthropic integration with PreCallAI!
