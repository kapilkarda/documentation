
1. Go to [your account page](https://www.deepl.com/account/summary) on DeepL.
2. Scroll down and copy `Authentication Key for DeepL API`.
3. Paste the key into the `Authentication Key` field in Qcall AI.
4. Write any screen name to be displayed in Qcall AI.
5. Click `Save`.
6. Start using DeepL integration with Qcall AI!
