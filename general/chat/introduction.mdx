## What is <span style={{color:"#ca8a04"}}>QCall AI Chat</span> ?

QCall AI Chat enables real-time, AI-powered conversations with your customers across multiple messaging channels. It brings the same intelligent voice assistant capabilities to text-based messaging, allowing you to engage customers through SMS, WhatsApp, and web chat.

## Key Features

- **Multi-Channel Support**: Engage customers via SMS, WhatsApp, and web chat from a single platform
- **AI-Powered Responses**: Leverage your existing assistant scripts and knowledge base for intelligent chat responses
- **Live Handoff**: Seamlessly transfer conversations from AI to human agents when needed
- **Automated Follow-ups**: Send automated messages after calls or based on trigger events
- **CRM Integration**: Sync chat conversations with your existing CRM (HubSpot, Salesforce, Zoho, GoHighLevel)

## How to Access Chat

Login into the [dashboard](https://app.qcall.ai/admin/dashboard) and open the Chat section.

## Chat Channels

| Channel | Description | Status |
|---------|-------------|--------|
| SMS | Send and receive text messages using your dialer phone number | Available |
| WhatsApp (Zeplo) | Send WhatsApp messages using approved templates via Zeplo integration | Available |
| Web Widget | Embed a chat widget on your website for visitor engagement | Available |
